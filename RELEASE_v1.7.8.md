# Release v1.7.8

## Instalador Windows

Arquivo: `Instalador_PainelTribuna_v1.7.8.exe`

## Novidades desta versão

- Scripts `install.bat` e `run.bat` alinhados ao ambiente `.venv`
- `requirements.txt` com dependências do projeto (PySide6, Flask, etc.)
- Suporte a `py -3` quando `python` não está no PATH
- `multiprocessing.freeze_support()` para build com PyInstaller

## Instalação

1. Baixe o `.exe` desta release
2. Execute o instalador como administrador (se solicitado)
3. Opcional: instale o driver USB CH340 durante a instalação
4. Inicie pelo atalho **Painel de Controle Tribuna**
