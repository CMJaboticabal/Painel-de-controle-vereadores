# Release v1.7.9

## Instalador Windows

Arquivo: `Instalador_PainelTribuna_v1.7.9.exe`

## Novidades

- Seleção do monitor para a **tela do público** (padrão: Monitor 2)
- Suporte a 3 ou mais monitores na configuração da sessão
- Botão para atualizar a lista de monitores conectados
- Reposicionamento automático da tela ao salvar configurações

## Instalação

1. Baixe o `.exe` desta release
2. Execute o instalador (como administrador, se solicitado)
3. Opcional: instale o driver USB CH340 durante a instalação
4. Em **Administração → Tela Secundária**, confira o **Monitor do Público**
5. Inicie pelo atalho **Painel de Controle Tribuna**
