Sistema de Automação
Para Tribuna de Áudio

Guia detalhado de componentes e especificações para montagem profissional com supressão de ruídos.

Especificações de Compra Diagrama do Capacitor

Objetivo do Sistema

Este sistema automatiza o corte e liberação do áudio em uma tribuna oficial. Utilizando um microcontrolador Arduino e relés, ele garante que o microfone esteja ativo apenas durante o tempo permitido pelo operador.

Principais benefícios:

  • Precisão no controle de tempo.
  • Segurança operacional (Fail-Safe).
  • Eliminação de ruídos de comutação (Pop).

Confiabilidade do Sistema Fail-Safe

A lógica Normalmente Fechado (NC) garante que o áudio funcione mesmo sem energia no sistema de controle.

Guia de Compra: Componentes Específicos

Para um áudio limpo, a escolha do hardware é fundamental. Abaixo estão as especificações exatas para aquisição.

📟

Microcontrolador

Recomendado para estabilidade e conexão USB segura.

  • Modelo: Arduino Uno R3 ou Nano V3.
  • Chip: ATMega328P.
  • Cabo: USB tipo A/B (Uno) ou Mini-USB (Nano).
🔌

Módulo Relé Profissional

Essencial para isolar o ruído elétrico do áudio.

  • Tipo: 2 Canais com Optoacoplamento.
  • Tensão: Bobina de 5V DC.
  • Contatos: 10A / 250VAC (Suporta NC/NF).
💎

Capacitor de Filtro

O segredo para eliminar o "estalo" (pop) da tribuna.

  • Tipo: Cerâmico Multicamadas.
  • Valor: 100nF (Código impresso: 104).
  • Tensão Mín.: 50V (Standard).

💡 Conceito Fail-Safe

Usamos o terminal NC (Normalmente Fechado). Isso significa que, sem energia, o relé fecha o contato e deixa o som passar. Se o sistema travar ou desligar, você não perde o áudio. O comando de "Corte" ativa o relé para *abrir* o circuito.

2. Instruções de Soldagem

  1. A Fios Diretos (Bypass)

    Solde um cabo direto do Pino 1 (Entrada) ao Pino 1 (Saída).
    Faça o mesmo para o Pino 3 (Entrada) ao Pino 3 (Saída).
    Estes fios não tocam no relé.

  2. B O Corte (Pino 2)

    O fio do Pino 2 deve ser seccionado (cortado) e desviado para o relé:

    XLR Entrada (Pino 2) ───🡢 COM (Centro)
    XLR Saída (Pino 2) 🡠─── NC (Fechado)
    Terminal NO (Aberto) ....... VAZIO
Dica Pro: Mantenha os fios do relé o mais curtos possível para evitar zumbidos (hum) de rede elétrica, já que o Pino 2 carrega o sinal vivo balanceado.

Eficácia do Filtro Cerâmico 104

O capacitor cerâmico de 100nF (104) é escolhido por sua alta velocidade de resposta.

Ele atua drenando a energia transiente que saltaria entre os contatos COM e NC no momento da abertura, garantindo que o som seja cortado de forma "suave" eletricamente.

Por que Cerâmico?

Capacitores eletrolíticos não servem aqui por serem polarizados e mais lentos. O cerâmico tipo "disco" ou "multicamada" é o padrão industrial para filtros de ruído.

Programação do Arduino (.ino)

Após concluir a ligação elétrica, grave o firmware do projeto no Arduino para habilitar o controle automático de áudio pelos relés.

Arquivo correto do firmware

Use este arquivo do projeto:

arduino_relay_control/arduino_relay_control.ino

Passo a passo (Arduino IDE)

  1. Conecte o Arduino ao computador via USB.
  2. Abra a Arduino IDE.
  3. No menu Arquivo, abra o arquivo .ino indicado acima.
  4. Em Ferramentas → Placa, selecione o modelo correto (ex.: Arduino Uno/Nano).
  5. Em Ferramentas → Porta, selecione a porta serial do Arduino.
  6. Clique em Verificar (✓) para compilar.
  7. Clique em Carregar (→) para gravar no dispositivo.
  8. Aguarde a mensagem "Upload concluído".

Validação rápida após gravação

  • Abra o sistema no computador e verifique se o status do Arduino aparece como conectado.
  • Use os botões de teste no painel administrativo para abrir/cortar áudio.
  • Se não conectar, revise placa/porta selecionadas na IDE e driver USB do Arduino.
Nota: Em placas Nano, se o upload falhar, teste também Ferramentas → Processador → ATmega328P (Old Bootloader).

Dicas de Compra e Diagnóstico

🛒 Onde Comprar?

Componentes comuns encontrados em lojas de eletrônica.

Sugestão: Procure kits de "Iniciante Arduino". O capacitor 104 costuma vir em conjuntos de resistores/capacitores.

🔋 Fonte de Energia

O Arduino pode falhar se a porta USB não fornecer corrente suficiente para os dois relés.

Solução: Utilize uma fonte de 5V 2A externa (tipo carregador de celular) conectada ao Arduino via USB para maior torque mecânico nos relés.

📦 Proteção Física

Ruídos externos podem ser captados pelos fios soltos.

Solução: Acomode tudo em uma Caixa Patola. O plástico isola os contatos elétricos e evita curto-circuitos acidentais na tribuna.
